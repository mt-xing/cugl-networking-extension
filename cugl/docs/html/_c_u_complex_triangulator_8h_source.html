5 // This module is a factory for a Constrained Delaunay triangulator. This
6 // is distinct from the normal Delaunay triangulator in that it can handle
7 // complex polygons (polygons with holes, but not self-crossings). It is
8 // also much more heavy-weight than the other triangulators because it is
9 // built on top of the popular poly2tri library
10 //
11 // https://github.com/jhasse/poly2tri
12 //
13 // Because math objects are intended to be on the stack, we do not provide
14 // any shared pointer support in this class.
