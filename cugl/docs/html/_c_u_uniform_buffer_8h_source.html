5 // This module provides a uniform buffer for sending batched uniforms to a shader.
6 // Unlike a vertex buffer, uniform buffers are optional (you can set the uniforms
7 // directly in the shader). Uniform buffers are solely a performance optimization.
8 // In our tests, they only provide a win when (1) there are a large number of
9 // uniforms being passed to the shader, (2) these uniforms can be loaded at the
10 // start of the frame, and (3) shader updates are done through offset management.
11 //
12 // This class uses our standard shared-pointer architecture.
13 //
14 // 1. The constructor does not perform any initialization; it just sets all
15 // attributes to their defaults.
16 //
17 // 2. All initialization takes place via init methods, which can fail if an
18 // object is initialized more than once.
19 //
20 // 3. All allocation takes place via static constructors which return a shared
21 // pointer.
