3 // Cornell University Game Library (CUGL)

5 // This module provides the class for representing a 2D OpenGL texture. This
6 // class also provides support for texture atlases. Any non-repeating texture
7 // can produce a subtexture. A subtexture wraps the same texture data (and so
8 // does not require a context switch in the rendering pipeline), but has
9 // different start and end boundaries, as defined by minS, maxS, minT and maxT
10 //
11 // This class uses our standard shared-pointer architecture.
12 //
13 // 1. The constructor does not perform any initialization; it just sets all
14 // attributes to their defaults.
15 //
16 // 2. All initialization takes place via init methods, which can fail if an
17 // object is initialized more than once.
18 //
19 // 3. All allocation takes place via static constructors which return a shared
20 // pointer.
