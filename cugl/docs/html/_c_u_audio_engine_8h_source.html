3 // Cornell University Game Library (CUGL)
4 //
5 // This module is an attempt to combine the power of a modern DSP mixer graph
6 // with a simple 2000-era interface. Like the legacy engines, it provides a
7 // a flat slot-based structure for playing sounds, and controlling the fade
8 // and pan of each slot. It also provides support for music queues.
9 //
10 // However, unlike legacy engines, this engine is not limited to playing
11 // music samples. It also allows you to attach and play any arbitrary audio
12 // node. For example, you could combine multiple simultaneous source together
13 // and play them together to provide vertical layering. However, these
14 // nodes are still wrapped in a top-level fader to prevent clicking like
15 // you can get in OpenAL engines.
