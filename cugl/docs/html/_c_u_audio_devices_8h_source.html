5 // This module is a singleton for managing audio devices in the game engine.
6 // This singleton can support multiple input and output devices for complex
7 // filter graphs. This class is for when developers needs direct access
8 // to the audio devices. In most cases, developers can use AudioEngine
9 // instead, which is built on top of this device manager.
10 //
11 // Because this is a singleton, there are no publicly accessible constructors
12 // or intializers. Use the static methods instead.
