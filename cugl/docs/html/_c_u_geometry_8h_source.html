5 // This module provides an very robust enum that that categorizes polygon
6 // geometry. In earlier versions of the CUGL, this type was built into the
7 // Poly2 class. However, we realized that there is a lot of semantic
8 // information in this type that is separate from the polygon that it is
9 // attached to. Therefore, we have made it into its own separate class.
10 //
11 // Because math objects are intended to be on the stack, we do not provide
12 // any shared pointer support in this class.
