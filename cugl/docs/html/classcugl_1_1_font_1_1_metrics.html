Detailed Description

This class is a simple struct to store glyph metric data

A glyph metric stores the bounding box of a glyph, pluse the spacing information around it. The bouding box is offset from an origin, and the advance the distance to the next glyph origin. For more information, see http://jcatki.no-ip.org:8080/SDL_ttf/SDL_ttf.html#SEC38
