◆ receive()

void cugl::CUNetworkConnection::receive ( const std::function< void(const std::vector< uint8_t > &)> &  dispatcher)

Method to call every network frame to process incoming network messages.

A network frame can, but need not be, the same as a render frame.

This method must be called periodically EVEN BEFORE A CONNECTION IS ESTABLISHED. Otherwise, the library has no way to receive and process incoming connections.

Parameters
dispatcherFunction that will be called on every byte array sent by other players.
